# layout布局
### - LinnerLayout
* 可以设置水平或者垂直方式摆放组件

```
android:gravity:指定如何在该对象中放置次对象的内容（X/Y坐标值）
android:orientation:设置它内容的对齐方向（横向/竖向）
android:layout_width="wrap_content" 虽内容改变
android:layout_height="match_parent"填充父容器
android:layout_weight:权重值默认为0
```
